Start Coding: HTML

To start coding, you have to specify a doctype.
A doctype is a piece of code where you tell the computer what type of HTML your coding.
On this website, I will use html5.
When you specify the doctype for html5, you type <!DOCTYPE html>
After that, you drop down a line and type <html>
For all tags, you must close it so after <html> type </html> to close the tag
<!DOCTYPE html>
<html></html>
After that, take a line down.
<!DOCTYPE html>
<html>
</html>
Then you open the tags
<!DOCTYPE html>
<html>
People usually add tabs after they open 2 tags.
Or if they have a text editor that does it for them they don't.
Then type head in a tag. The code in the head element is not visible on the website.
<!DOCTYPE html>
<html>
<head>
</head>
</html>
Inside head, we will type <title></title>
<!DOCTYPE html>
<html>
<head>
<title></title>
</head>
</html>
Inside the title, we specify the name of the webpage (which will be shown at the top of the tab).
<!DOCTYPE html>
<html>
<head>
<title>Document</title>
</head>
</html>
